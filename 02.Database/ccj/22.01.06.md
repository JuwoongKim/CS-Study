####NoSQL
배경
- 비정형 같은 방대한 데이터를 미리 스키마 정의하는 것이 거의 불가능

구조
- 문서 데이터베이스
    - Json 객체와 비슷한 문서에 데이터를 저장
    - MongoDB
- 키-값 데이터베이스
    - Redis, DynanoDB
- 와이드 컬럼 스토어
    - 테이블, 행 및 동적 열에 데이터 저장
    - Cassandra, HBase
- 그래프 데이터베이스
    - 노드와 에지에 데이터를 저장
    - Neo4j, JanusGraph
